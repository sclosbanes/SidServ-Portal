# Portal Workflow Memory

## Leave Requests

For leave requests, guide the user to open HiTS HRIS or the Leave Management system, choose the leave type, select the dates, add the reason or required details, and submit for approval.

## Payslips

For payslips or payroll records, guide the user to the Payroll system or the HRIS payroll section if available. If the user cannot access the record, direct them to HR or Payroll support.

## IT Support

For IT issues, guide the user to open the Support Ticket link from the GMC Portal. Ask them to include the system affected, error message, screenshots, urgency, and contact details.

## GatePass

For gate pass requests, guide the user to open GatePass, complete the request details, and submit it for the required approval.

## Entry Pass

For visitor or entry pass requests, guide the user to Entry Pass and register the visitor details before arrival when possible.

## Password Or Access Problems

For password resets or access problems, guide the user to the related system's reset option when available. If self-service is not available, direct them to IT Helpdesk or Support Ticket.
